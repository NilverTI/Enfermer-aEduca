# Enfermería Educa

Plataforma educativa con contenidos de salud útiles, claros y accesibles.

## Sobre el Proyecto
Desarrollamos material didáctico y visual para mejorar la comprensión de temas de salud fundamentales, democratizando el conocimiento con un diseño moderno, ágil y atractivo (Premium UI/UX).

**Tecnologías:**
- HTML5 Semántico
- CSS Variables, Grid Bento Layout y Glassmorfismo
- JavaScript Nativo (Vanilla)

**Desarrollado para la Comunidad Médica y Estudiantil.**
